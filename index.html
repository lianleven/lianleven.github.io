macOS装机必备

发表于 2018-09-19 | 更新于 2018-09-20 | 分类于 资源和工具


macOS 设置指南

Xcode

Xcode是macOS的集成开发环境

安装 Xcode 命令行工具：

从App Store或Apple的网站下载并安装它。

Homebrew

macOS 缺失的软件包的管理器

安装 Homebrew，将命令粘贴到终端

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

FFmpeg

一个完整的跨平台解决方案，用于录制，转换和流式传输音频和视频。

安装 FFmpeg，将命令粘贴到终端

FFmpeg 详细使用看官方文档, 很强大的一个库

brew install ffmpeg

Homebrew-Cask

Homebrew-Cask 允许您通过命令行工具安装大型二进制文件。
例如:
您可以安装Google Chrome，Dropbox，VLC和Spectacle等应用程序。
不用再下载.dmg文件并将它们拖到您的Applications文件夹中！

使用 Homebrew-Cask 搜索应用和插件

$ brew cask search <package>
// 搜索 alfred 应用
$ brew cask search alfred

预览插件 Quick Look plugins

$ brew cask install \

qlstephen \
qlmarkdown \
quicklook-json \

quicklook-csv \

alfred

Alfred 可通过热键，关键字，文本扩展等提高您的效率。
搜索您的Mac和网络，并通过自定义操作提高控制Mac的效率。

具体使用配置网上很多教程，这里就不详细介绍了。

pdftk

PDFtk是一个用PDF文档处理日常事务的简单工具。
具体使用请看官方文档

sublime-text

用于代码，标记和散文的复杂文本编辑器

appcleaner

AppCleaner是一个小应用程序，允许您彻底卸载不需要的应用程序。
安装应用程序会不必要地使用硬盘空间在整个系统中分发许多文件。
AppCleaner找到所有这些小文件并安全地删除它们。
只需将应用程序拖放到AppCleaner窗口即可。
它将找到相关文件，您可以通过单击删除按钮删除它们。
